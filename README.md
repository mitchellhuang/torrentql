# TorrentQL

[![Build Status](https://drone.huang.mx/api/badges/mitchellhuang/torrentql/status.svg)](https://drone.huang.mx/mitchellhuang/torrentql)

Download and seed torrents on-demand.

## Packages

* nextjs (frontend)
* graphql (backend)
* deploy (ci)

## Install and link packages

```
yarn install
yarn bootstrap
```
